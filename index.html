What is Next OS?

Next OS is an operating system based on Ubuntu. Next OS aims to be a quick and powerful OS that can do daily tasks easily.

Where can I get Next OS?

Right now, you are not able to download Next OS, as it is in a very early alpha stage. We hope to release the open beta of Next OS sometime in 2023-2024

What desktop enviroment will Next OS use?

Right now, Next OS uses the MATE desktop enviroment, though this may change in the future.
